Listas em HTML5

1. Introdução

Listas são essenciais para estruturar e apresentar informações de forma organizada e acessível. Existem três tipos principais em HTML5: <ul> (não ordenada), <ol> (ordenada) e <dl> (de definição).

2. Lista Não Ordenada (<ul>)

Usada quando a ordem dos itens não importa, como em listas de compras ou menus.

Exemplo:

Minha Lista de Compras

  • Pão
  • Leite
  • Ovos
  • Café

3. Lista Ordenada (<ol>)

Usada quando a ordem importa, como em tutoriais ou rankings.

Exemplo:

Como Fazer um Bolo

  1. Misture os ingredientes secos.
  2. Adicione os líquidos.
  3. Bata até ficar homogêneo.
  4. Asse por 40 minutos.

Top 3 Filmes (com tipo e início personalizados)

  1. Filme A
  2. Filme B
  3. Filme C

4. Lista de Definição (<dl>)

Cria pares termo-definição. Útil para glossários e FAQs.

Exemplo:

Glossário de Termos Web

HTML
Linguagem de Marcação de Hipertexto, usada para estruturar conteúdo na web.
CSS
Folhas de Estilo em Cascata, usadas para estilizar a aparência de documentos web.
JavaScript
Linguagem de programação para tornar páginas web interativas.

5. Listas Aninhadas

Uma lista pode conter outra lista dentro de um item <li>, útil para menus e hierarquias.

Exemplo:

Menu Principal

  • Home
  • Produtos
    • Eletrônicos
    • Roupas
      • Masculino
      • Feminino
  • Contato

6. Importância das Listas

  • Organizam o conteúdo de forma lógica
  • Melhoram a acessibilidade com leitores de tela
  • Ajudam no SEO (melhor ranqueamento)
  • Facilitam a manutenção do código
  • Permitem personalização via CSS

Conclusão

Usar corretamente <ul>, <ol> e <dl> ajuda a criar páginas bem estruturadas, acessíveis e mais fáceis de manter.
